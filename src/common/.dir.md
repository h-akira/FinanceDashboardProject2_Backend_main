# Backend/main/src/common/

- `config.py`: 環境変数の一元管理。新しい環境変数はここに追加し、`template.yaml` の Environment と `README.md` の環境変数表も更新すること
- `auth.py`: JWT クレームからのユーザー識別ヘルパー。認証方式を変更した場合は `Backend/main/docs/technical_policies.md` も更新すること
- `exceptions.py`: AppError を基底とするエラー階層。新しいエラー種別はここにサブクラスを追加する。`app.py` のエラーハンドラが AppError を捕捉するため、サブクラスであれば自動的に処理される
