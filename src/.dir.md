# Backend/main/src/

- Lambdalith 構成（単一 Lambda）。レイヤーは routes → services → repositories の一方向依存
- フレームワークは aws-lambda-powertools（Router, Logger, Tracer）
- 環境変数は `common/config.py` で一元管理。コード内でのハードコードは禁止
- エラーは `common/exceptions.py` の AppError サブクラスを使用する
- 技術方針は `Backend/main/docs/technical_policies.md` を参照
- `custom_chart_sources.json` を変更した場合は `bin/validate_config.py` で検証し、`services/custom_chart_service.py` への影響を確認し、`Backend/main/docs/data_policies.md` も更新すること
