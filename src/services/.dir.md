# Backend/main/src/services/

- ビジネスロジック層。DynamoDB アクセスは repositories 経由で行う（直接 boto3 を呼ばない）
- 外部 API（FRED, yfinance）からのデータ取得・加工もここで行う
- DynamoDB の格納済みデータと動的データの結合ロジックを担う
- エラーは `common/exceptions.py` の AppError サブクラスを送出する
- データポリシーは `Backend/main/docs/data_policies.md` を参照
- 新サービス追加時は `routes/` からの呼び出しも追加すること。入出力変更時は対応するルートとテストも更新すること
- データ取得ロジックを変更した場合は `Backend/main/docs/data_policies.md` も更新すること
