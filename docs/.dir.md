# Backend/main/docs/

- `openapi_main.yaml` は親リポジトリ（`docs/openapi_main.yaml`）から同期されるファイル。絶対に編集しないこと
- `technical_policies.md`: 技術スタック・フレームワーク・認証・デプロイ等の技術方針。技術選定や方式を変更した場合は更新すること
- `database_design.md`: DynamoDB シングルテーブル設計（キーパターン・アクセスパターン）
- `data_policies.md`: 外部データソースの取得・サンプリング・境界日の定義
- 影響範囲: `database_design.md` や `data_policies.md` を変更した場合、`src/repositories/`・`src/services/`・`bin/` の実装との整合性を確認すること
